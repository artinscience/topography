# artinscience · topography

Overhead **topographic / terrain art** — contour and relief drawn from real
elevation data as pen-plotter line work. Multi-pen by elevation band; hillshade
rendered as hatching; ridgeline ("joy plot") styling.

**Subjects:** mountains, national parks, coastlines, lakes & watersheds, the
ocean floor — and, with planetary elevation models, the terrain of Mars and the
Moon.

**Built from** public DEM data (SRTM, USGS 3DEP, Copernicus DEM, OpenTopography)
via QGIS; OSM for water & coastlines.

---

Part of the artinscience project — generative pen-plotter art.
